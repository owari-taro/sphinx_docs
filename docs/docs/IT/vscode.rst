複数立ち上げたときに色分けしたい
==================================
.vscode/settings.jsonに以下を記述する(`qita <https://qiita.com/deren2525/items/6bc099ae8c05e3076055>`__)


::

    {
    "workbench.colorCustomizations": 
    {
    "editorGroup.border": "#d62323",
    
    "sideBar.border": "#ef4516",

    "activityBar.background":"#2d09cf"
    }
    }
