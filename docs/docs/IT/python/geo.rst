gdal
===========
gdal2tilesインストールする前には下記を実行する必要あり
:: 

    sudo apt update
    sudo apt-get install libgdal-dev
    sudo apt-get install build-essential libssl-dev libffi-dev python3-dev
    sudo apt install gdal-bin
