第4章 インターネット接続¶

  • サブネットにルートテーブル設定しないとvpcに紐づくメインルートテーブルが適用される

  • VPC内のすべてのサブネットに共通化したいならメインルートテーブルを変更うすればよい

  • ルートテーブルを作成した直後はVPC内にしか通信が通信できなくなっている

  • ec2にはprivateIPが割り振られている
